Function Name: perfectSquare

Inputs  (1): -(double) a vector of integers
Outputs (2): -(double) a vector of all perfect squares from input vector
                   -(double) the number of perfect squares

Function Description:
Write a function called perfectSquare that finds all of the perfect squares in a vector. A perfect square is a number whose square root is an integer. The first output is a vector that contains only the perfect squares from the vector. The second output is how many perfect squares are in the original vector.

Test Case:
vec = [5 2 4 9 2];
[squares count] = perfectSquare(vec);
   squares => [4 9]
   count => 2

function [squares, count] = perfectSquare(vec)

square_root = sqrt(vec);

squares = [vec(mod(square_root,2)==0) | vec(mod(square_root,1)==0)];

count = length(squares);

Outputs (1): -(double) The modified array
                 
               
Function Description:
Write a function called arrayMath that will edit the array in the following order. Multiply all odd NUMBERS by 2. Substract 5 from all of the numbers in even INDEXED columns. Then, replace all negative numbers with 0 and return the modified array.

Test Case:
    arr = [2 2 4
         1 3 5
         7 8 9]

out = arrayMath(arr)
    out = [2  0  4
           2  1 10
           14 3 18]

function [modarr] = arrayMath (arr)

b = arr.*2

arr(mod(arr,2) == 1) = b((mod(arr,2) == 0))

arr(2:2:end) = arr(2:2:end) - 5

arr(arr < 0) = 0

modarr = arr

function [alternating] = camelCase(str)
